Research

Wi-PRO: Accurate Indoor PNT using FTM with Multipath Suppression

802.11mc (FTM Ranging) has been standardized in Wi-Fi for a few years now, but we see very few commercial products using it. This is because most open FTM implementations have extremely poor empirical performance indoors, due to multipath. In this work, I show that you can use CIR measurements exposed by the ESP32 with a simple leading-edge detection algorithm to cancel the effect of multipath, and obtain a 3-4x improvement in range error over the ESP32's implementation. I have also released a toolbox [Code] along with the paper [Paper] so others can experiment with wireless localization systems on the ESP32.

Wireless Synchronization

In collaboration with Meta's Time Appliance Project (subsidiary of the Open Compute Project,) I developed a wireless synchronization platform that uses LoRa packets to accurately time- and frequency- lock two boards. For this project, we implemented a VCO+Clock Servo on an FPGA and then developed an interesting SDR-like architecture using the SX1256 radio fronend, and implement our own LoRa baseband receiver inside tge FPGA in order to get I/Q samples timestamped with respect to the syncrhonized clock. This lets us obtain instantaneous clock phase offsets constrained to within 10-20 nanoseconds. Work is currently seeking publication.

Wi-Fi Localization

During my undergrad I worked on algorithms to help robots navigate indoors using Wi-Fi beacon signals generated by access points. I used an antenna array to compute angle-of-arrival of the beacon packets at the robot and then several SLAM algorithms to reconstruct a map of the environment containing the access point location. These AP direction measurements help accurately reconstruct a map and reduce the computational complexity of navigation.
